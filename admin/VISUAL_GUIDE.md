## 🔧 การแก้ไขปัญหา

### ปัญหา: เข้าสู่ระบบไม่ได้
```
❌ ข้อผิดพลาด: Invalid credentials
✅ วิธีแก้: ตรวจสอบ .env.local
   - ADMIN_EMAIL=admin@greenbluerestbangkok.com
   - ADMIN_PASSWORD_HASH=ถูกต้อง
```

### ปัญหา: อัปโหลดไฟล์ไม่ได้
```
❌ ข้อผิดพลาด: Upload failed
✅ วิธีแก้: ตรวจสอบ
   - ขนาดไฟล์ ≤ 8MB
   - ประเภทไฟล์: PNG, JPG, WebP, GIF
   - GitHub token มี permission
```

### ปัญหา: บันทึกเนื้อหาไม่ได้
```
❌ ข้อผิดพลาด: Save failed
✅ วิธีแก้: ตรวจสอบ
   - JSON format ถูกต้อง
   - Path ไม่มีอักขระพิเศษ
   - GitHub token ถูกต้อง
```

## 📊 Flow การใช้งาน

### การสร้างบทความใหม่
```
1. Dashboard → สร้างเนื้อหาใหม่
2. เลือก Markdown/JSON
3. กรอกข้อมูล
4. บันทึก
5. เปลี่ยน status เป็น published
6. ตรวจสอบในเว็บไซต์หลัก
```

### การอัปโหลดรูปภาพ
```
1. Dashboard → จัดการสื่อ
2. ลากไฟล์มาวาง
3. รอให้อัปโหลดเสร็จ
4. คัดลอก path
5. ใช้ path ในบทความ
```

### การแก้ไขเนื้อหา
```
1. Dashboard → ดูเนื้อหาทั้งหมด
2. คลิกแก้ไข
3. แก้ไขข้อมูล
4. บันทึก
5. ตรวจสอบการเปลี่ยนแปลง
```
